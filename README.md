# Pharmacy Location Optimization QR Website

Repository-ready static website for the graduation project.

**GitHub Pages target URL:** https://minldmrc.github.io/Graduation-Project-Of-Industrial-Engineering-/

## Contents
- `index.html` — final responsive website
- `assets/poster_qr_github_pages.png` — QR code used inside the website
- `qr_github_pages.png` — repository-level QR code file
- `assets/` — poster, team photos and visual outputs
- `maps/` — interactive GIS maps
- `downloads/` — source code, report and Excel outputs
- `.nojekyll` — GitHub Pages compatibility file

## Deployment
Upload all files to the repository root and enable GitHub Pages from **Settings → Pages → Deploy from a branch → main / root**.
